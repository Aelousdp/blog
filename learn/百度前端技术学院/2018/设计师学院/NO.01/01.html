No.1 - 制作一个简单的菜单动画效果

课程目标：

通过大量练习，来学习布局的各种方式。

正式任务：

  1. 任务目的
    • 熟悉 CSS3 过渡子属性
    • 通过 JavaScript 触发过渡效果
    • 理解语义化，合理地使用 HTML 标签来构建页面
  2. 任务描述
    • 通过 CSS transition 实现demo效果
    • 每次点击【切换样式】按钮的时候，出现视频 demo 所示的动画效果。
    • 研究一下，如何通过修改贝塞尔曲线，让这个动效更贴近自然。
  3. 任务注意事项
    • CSS transition 的各项子属性（时间曲线）等详细值可以自由定义；
    • 注意浏览器中的兼容性；
    • HTML 及 CSS 代码结构清晰、规范；
    • 代码中含有必要的注释；
