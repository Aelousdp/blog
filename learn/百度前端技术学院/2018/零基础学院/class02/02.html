验证：

  • 问：HTML是什么，HTML5是什么

    答：HTML是超文本标记语言，超文本指的就是页面内可以包含图片、链接、音乐、程序等元素。 HTML5 将成为 HTML 、 XHTML 以及 HTML DOM 的新标准。

  • 问：HTML元素标签、属性都是什么概念？

    答：HTML 元素指的是一整个闭合的标签，即：开始标签+内容+闭合标签。 HTML 属性是标签所拥有的，为元素提供更多的信息，例如：name="张三"。属性在 HTML 元素的开始标签中规定。

  • 问：文档类型是什么概念，起什么作用？

    答：文档声明帮助浏览器正确地显示网页。 HTML 也有多个不同的版本，只有完全明白页面中使用的确切 HTML 版本，浏览器才能完全正确地显示出 HTML 页面。

  • 问：meta标签都用来做什么的？

    答：<meta>标签提供关于 HTML 文档的元数据。 它不会显示在页面上，但是对于机器是可读的。 可用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他 web 服务。

  • 问：Web语义化是什么，是为了解决什么问题

    答：Web语义化是指使用语义恰当的标签，使页面有良好的结构，页面元素有含义，能够让人和搜索引擎都容易理解。

  • 问：链接是什么概念，对应什么标签？

    答：HTML通过链接与网络上的另一个文档相连接。对应的标签是 <a> 标签。

  • 问：常用标签都有哪些，都适合用在什么场景

    答：常用标签：<html>：定义HTML文档；<title>：定义文档的标题；<h1>-<h6>：一级标题-六级标题；<p>：定义段落；<div>：定义文档中的块；<a>：定义锚链接；<b>：定义粗体；<img>：定义图像；<input>：定义输入；<style>：定义文档的样式信息；<script>：定义客户端脚本；

  • 问：表单标签都有哪些，对应着什么功能，都有哪些属性

    答：
    • <input>元素：根据不同的 type 属性，可以变化为多种形态；
    • <select>元素：元素定义下拉列表；<option>元素：定义待选择的选项。
    • <textarea>元素：定义多行输入字段（文本域）。
    • <button>元素：定义可点击的按钮。

  • 问：ol, ul, li, dl, dd, dt等这些标签都适合用在什么地方，举个例子

    答：<ol>+<li>：有序列表。用于简单的项目列表。比如，水果：1、苹果 2、香蕉 3、葡萄，并不显示详细信息。<ul>+<li>：无序列表。使用同有序列表，只是没有数字标识。<dl>+<dd>+<dt>：自定义列表。适用于对列表信息进行简单描述，例如：水果：苹果，香蕉；蔬菜：西红柿，黄瓜;
